# SCALL_TECNM-CLN
Aplicacion para Sistema de captación y cloración de agua de lluvia

# Representación del modelo
El sistema se compone de dos unidades principales de almacenamiento y procesamiento:
•	Tanque de Almacenamiento T2: Representa el depósito superior con un área transversal A2 y un nivel de agua variable h2. Recibe un flujo de entrada Q12 (simulando lluvia).

•	Tanque de Mezcla y Cloración T3: Es la unidad inferior donde ocurre el proceso de desinfección. Tiene un área A3 y su nivel h3 es monitoreado para el control de volumen.

•	Actuadores de Flujo (valv, valv1): Válvulas solenoides que regulan el caudal transferido Q23 entre tanques y la descarga final Q3 hacia el usuario.

•	Sistema de Dosificación (Bomba, Ccloro): Una bomba peristáltica encargada de inyectar cloro (Qcloro) desde una solución madre con concentración fija (Ccloro) hacia el Tanque 3.
